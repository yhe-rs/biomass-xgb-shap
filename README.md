
# Background
This is the python codes for paper: Forest Aboveground Biomass Estimation through Integration of Sentinel-2 and PALSAR-2 Time Series: Assessing Models Trained on GEDI and Field Inventory Benchmarks. (https://doi.org/10.1016/j.isprsjprs.2026.04.022)
* Evaluate the impact of temporal feature representations (e.g., static annual means, quarterly averages and annual summaries) and sensor configurations (Sentinel-2, PALSAR-2, and their fusion) on AGB prediction.
* Compare model performance using GEDI-derived biomass products and field inventory benchmarks.
* Assess model robustness across environmental gradients: elevation, slope, and forest density.
